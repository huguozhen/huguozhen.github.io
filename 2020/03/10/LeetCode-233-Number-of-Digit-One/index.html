233. Number of Digit One

Hard

  • 对于 n = 1230567，分析数字 1 在千位上出现的次数，可能的形式包括 1221999 ~ 1221000 ，1211999 ~ 1211000 ，… … ，0001999 ~ 0001000 ，共 123 * 1000 次；
