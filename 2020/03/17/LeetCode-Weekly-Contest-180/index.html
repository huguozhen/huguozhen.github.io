1382. Balance a Binary Search Tree

Medium

Description

Go to LeetCode

Solution 1

考虑到一个有序数组转化为平衡二叉树是容易的（首先将中间的元素作为根节点，再利用分治法递归处理两边即可），而二叉排序树的中序遍历序列就是一个升序数组，因此先对 BST 进行中序遍历得到升序数组，再根据有序数组建立 BBST 即可。注意数组直接存储节点指针比较省空间。

此方法时间复杂度为和空间复杂度均为 $O(n)$ 。

Solution 2

实际上，有所谓 DSW 算法能够在 $O(n)$ 时间和 $O(1)$ 空间内对二叉排序树进行平衡化，详见下列资料：

Day–Stout–Warren algorithm - Wikipedia
