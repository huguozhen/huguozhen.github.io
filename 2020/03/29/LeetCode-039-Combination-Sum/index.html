039. Combination Sum

Medium

Description

Go to LeetCode

Solution

该问题具有非常明显的递归结构：在 candidates 中搜索 target 的所有 combination 等价于先选择每个可能的 candidates[i] 作为 combination 中的一个数，然后在 candidates 中搜索 target-candiates[i] 的所有 combination 。故可用回溯法递归地解决该问题。

需要注意的一点是如何去除重复的 combination ，其实只需要先将 candidates 排序，然后每一层递归均从上一层选择的元素开始向后遍历，这样就能够保证遍历到的 combination 是有序的，因而也是唯一的。
